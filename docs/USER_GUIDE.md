## Getting Started

### Logging In

1. Navigate to the A2 Intelligence platform URL
2. You'll see the login screen with two options:
   - **Continue with Google**: Use your Google account for single sign-on
   - **Email Login**: Enter your registered email and password
3. Click "Sign In" to access the platform

### Navigation

The platform uses a sidebar navigation on the left with the following sections:

- **ANALYTICS**: Core analysis modules
- **SCENARIOS**: Scenario management tools
- **PORTFOLIO**: Portfolio management features

### Nature Risk

Comprehensive nature-related risk analysis using TNFD LEAP methodology.

**Key Badges:**
- TNFD Framework compliant
- ENCORE Database integrated
- GBF Aligned

**Tabs:**

| Tab | Description |
|-----|-------------|
| Dashboard | Nature risk overview with KPIs |
| LEAP | LEAP Assessment Wizard (4-step process) |
| Portfolio | Portfolio-level nature risk analysis |
| Water Risk | Water stress analysis with interactive map |
| Biodiversity | Biodiversity site overlap analysis |
| ENCORE | Ecosystem services dependency explorer |
| GBF | Global Biodiversity Framework alignment |

#### LEAP Assessment Wizard

A 4-step process for comprehensive nature risk assessment:

1. **Locate**: Identify organization's interface with nature
   - Entity name and sector
   - Site information (name, country, coordinates)

2. **Evaluate**: Assess dependencies and impacts
   - Ecosystem service dependencies
   - ENCORE dependencies selection

3. **Assess**: Analyze risks under scenarios
   - Climate/nature scenario selection
   - Risk exposure sliders

4. **Prepare**: Develop mitigation strategies
   - Mitigation planning
   - TNFD disclosure readiness

#### Water Risk Map

Interactive Mapbox-powered map showing water stress locations:
- Color-coded markers by risk level
- Popup details on click
- Filter by country
- Scenario selection
- Map/List toggle view

## Scheduled Reports

Automate report generation and delivery.

### Creating a Scheduled Report:

1. Navigate to Portfolio Analytics > Scheduled tab
2. Click "New Schedule"
3. Configure:
   - Report Name
   - Module (portfolio_analytics, carbon_credits, etc.)
   - Format (PDF or Excel)
   - Frequency (daily, weekly, monthly, quarterly)
4. Save the schedule

### Managing Schedules:

- **Toggle Active/Paused**: Click the status toggle
- **Edit**: Update schedule parameters
- **Delete**: Remove the schedule

### Available Frequencies:
- Daily
- Weekly
- Monthly
- Quarterly
