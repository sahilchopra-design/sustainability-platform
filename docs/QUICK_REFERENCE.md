## Key Actions

### Run Analysis
1. Select Portfolio → 2. Choose Scenarios → 3. Set Time Horizon → 4. Click "Run"

### Export Data
1. Navigate to Module → 2. Click "Export" → 3. Choose PDF/Excel → 4. Download

### LEAP Assessment
1. Locate (entity info) → 2. Evaluate (dependencies) → 3. Assess (scenarios) → 4. Prepare (mitigation)

### Schedule Report
1. Portfolio Analytics → 2. Scheduled tab → 3. "New Schedule" → 4. Configure & Save

## Valuation Approaches

| Approach | Formula | Best For |
|----------|---------|----------|
| Direct Cap | Value = NOI / Cap Rate | Stabilized properties |
| DCF | NPV of 10-year cash flows | Changing cash flows |
| Cost | Land + (RCN - Depreciation) | Unique properties |
| Sales Comp | Adjusted comparable sales | Good market data |
